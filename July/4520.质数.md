# 4520. 质数

| 难度：**简单**                 |
| ------------------------------ |
| 时/空限制：1s / 64MB           |
| 总通过数：750                  |
| 总尝试数：1810                 |
| 来源：清华大学考研机试题       |
| 算法标签： 枚举 试除法 BFS DFS |

给定一个正整数 X，请你在 X 后面添加若干位数字（至少添加一位数字；添加的数不能有前导0），使得结果为质数，在这个前提下所得的结果应尽量小。

#### 输入格式

第一行包含一个整数 T，表示共有 T 组测试数据。

每组数据占一行，包含一个整数 X。

### 输出格式

每组数据输出一行结果，一个整数，表示所得的满足条件的最小质数。

#### 数据范围

1≤T≤100,
1≤X≤100。

#### 输入样例：

```
1
1
```

#### 输出样例：

```
11
```
